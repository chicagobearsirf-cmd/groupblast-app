# Getting Started with PostPilot

## One-time setup (takes about 5 minutes)

### Step 1: Install Node.js
PostPilot needs Node.js to run on your computer. If you don't have it yet:

1. Go to **https://nodejs.org**
2. Click the big green button (LTS version)
3. Open the downloaded file and click through the installer (just like installing any app)

### Step 2: Download PostPilot
1. Go to the PostPilot link your admin sent you
2. Click the green **Code** button, then **Download ZIP**
3. Unzip the file (double-click it) and put the folder somewhere easy to find, like your Desktop

### Step 3: Launch it
1. Open the **PostPilot** folder
2. Double-click **"Launch PostPilot.command"**
3. A black window will pop up — this is normal! The first time, it will take a few minutes to set itself up. Just wait.
4. Your browser will open automatically to PostPilot when it's ready.

## Every time after that

Just double-click **"Launch PostPilot.command"** again. It opens instantly.

**Keep the black window open while you're using PostPilot** — closing it stops the program. You can minimize it, just don't close it.

## Signing in

Enter your email on the login screen. We'll send you a link — click it and you're in. You'll stay signed in after that, so you won't need to do this every time.

## Need help?

Contact your admin if anything looks broken or you get stuck.
